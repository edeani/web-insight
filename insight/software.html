    • Software Y Dispositivos
    November 30, 2016

    El grupo INSIGHT ha desarrollado SOFTWARE para la automatización de metodologías de diagnóstico y predicción de diferentes fenómenos en medicina. Entre ellos, ha desarrollado metodologías para el diagnóstico de la dinámica cardiaca del adulto; ésta última evaluada con el Holter que se analiza a partir de medidas cuantitativas, objetivas y reproducibles del estado de la dinámica cardiaca, diferenciando normalidad de enfermedad y cuantificando el nivel de gravedad de las dinámicas patológicas, con base en la teoría de sistemas dinámicos, la probabilidad y la entropía. También se han desarrollado software que evalúan de manera unificada la función y estructura ventricular cardiaca izquierda a partir de exámenes como la angiografía coronaria y el ventriculograma, y un software que determina el estado de normalidad o restenosis de cortes trasversales de arterias coronarias, de aplicabilidad para modelos experimentales. En el área de neurociencias, se desarrollaron software para evaluar las etapas del sueño a partir de polisomnografías.

    En el área de la inmunología, se ha desarrollado un programa que automatiza una metodología predictiva del fenómeno de unión de péptidos al HLA clase II, que puede ser de utilidad para el desarrollo de vacunas, así como una metodología de predicción de unión de péptidos de malaria al glóbulo rojo. En el área de la predicción de epidemias, se han desarrollado metodologías para la evaluación y predicción de la dinámicas de malaria en Colombia con base en la teoría de la probabilidad, y en el área de la infectología, se desarrolló un software predictivo del número de linfocitos TCD4 en pacientes con HIV, que hacen innecesario el uso de citometría de flujo, disminuyendo costos y recursos y facilitando el seguimiento y la toma de decisiones terapéuticas para estos pacientes en zonas de bajos recursos. También se ha desarrollado un software para la administración de recursos empresariales, de utilidad para facilitar y automatizar sistemas y procesos en empresas.


    Software

    A continuación se encuentra el listado de software con registro de Propiedad Intelectual en la Dirección Nacional de Derechos de Autor:

    • Software que calcula los valores de proporciones de etapas de sueño y le asigna una evaluacion a cada etapa a partir de evaluaciones de tiempo obtenidas en polisomnografia.
    • Software que calcula la variabilidad y diferencia neta de dimensiones fractales de imágenes de ramificación coronaria izquierda.
    • Software que genera un conjunto de peptidos a traves de unión e intersección de conjuntos.
    • Software que calcula los microestados y las entropias de secuencias nonamericas de aminoacidos.
    • Software que calcula valores de probabilidad, entropia, macroestados y combinatoria de peptidos nonameros.
    • Software que calcula la armonía matemática intrínseca entre dimensiones fractales.
    • Software que calcula los grados de similitud para imágenes ventriculográficas.
    • Software de predicción de epidemia de malaria: calcula y dibuja un rango anual para la predicción de la epidemia de malaria.
    • Software que permite a partir de cálculos con la teoría de conjuntos establecer rangos en los que se puede encontrar la población de linfocitos CD4 para un paciente.
    • Software que predice la población de linfocitos TCD4, a partir del conteo de leucocitos y linfocitos del cuadro hemático, mediante la teoría de conjuntos y probabilidad.

    Dispositivos

    • Prototipo Holter para diagnóstico de la dinámica cardiaca neonatal con base en sistemas dinámicos, en desarrollo.
    • Equipo para diagnóstico de la dinámica cardiaca basado en la entropía proporcional de los sistemas dinámicos cardiacos.
